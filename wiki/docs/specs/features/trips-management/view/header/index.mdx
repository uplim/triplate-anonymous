# ヘッダーナビゲーション

## 外部仕様

### 構成

- [バックリンク](#バックリンク)
- [リロードボタン](#リロードボタン)
- [詳細ボタン](#詳細ボタン)
  - [削除ボタン](#削除ボタン)

### バックリンク

クリックすると、必ず[ホーム](/specs/features/trips-management/list/)へ遷移する．ブラウザの戻るボタンとは挙動が異なる．

### リロードボタン

クリックすると，ページをリロードする．

### 詳細ボタン

クリックすると下記要素を含んだモーダルを表示する．

#### 削除ボタン

クリックすると確認ダイアログを表示する．

- 「本当に削除してよろしいですか？」
  - 「OK」
    現在のたびを削除して[ホーム](/specs/features/trips-management/list/)
  - 「キャンセル」
